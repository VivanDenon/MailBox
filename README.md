# MailBox
Simple mail box client

Target platform: Windows 10 and lastest 

Using tecnologies: 
1) C# and WindowsForm 
2) STMP &amp; IMAP protocols
3) SSL-encryption
4) Async calls

You can log in, read/send messages, download/upload attachments, delete message and much more
